2.  板书（大三下）

相关比赛： 南京大学 - vivo Hackathon 编程马拉松大赛 2019
所获奖项： 二等奖（排名 3 / 36）
赛制介绍：4 人一小组，出题后连续 30 小时开发，每组 5 分钟时间展示。
题目：科技改变教育
项目介绍：我们组的立意是一个教师与学生共享白板的在线应用，教师在白板上板书（基于课件等），学生在自己的终端做笔记，在此基础上，我们还引入了学生笔记共享功能，通过板书时间轴，学生能任意融合别的学生公开的笔记到自己的笔记中，以此加深对课堂内容的理解。
分工责任：部分样式的编写； 基于 canvas 的自由书写系统以及相应移动端解决方案。
附图：github 提交记录

3.  个人博客项目（大三中到大三下）

本简历即依托于个人博客项目，点击页面左上角标志或者 这里 即可访问
项目地址： Github · StaticBlog
基本介绍：新的博客开始使用啦！
特点：这个博客较之前一版使用 express 做后端，sqlite 做数据库，Angular 做前端的博客简单得多，全部使用了静态文件（因为这样给学习和实验带来了非常高的自由度）。之后可能学习 SVG 等技术时可以很方便地容纳试验性的内容。
部署方面写了一个简单的部署脚本，将本地静态文件存放到服务器特定位置由 nginx 进行代理，所以本质上服务器端是本地博客项目的镜像，没有什么额外的部署成本（当然这也有很多缺点）。
简单的博文有相应的 markdown 转 html 脚本，所以如果没有特殊要求可以直接将 markdown 笔记转化成静态页面部署。
附图：个人博客项目结构和正在编写的简历页面
